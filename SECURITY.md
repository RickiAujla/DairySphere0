# Security Policy

## Supported Versions
We only patch security vulnerabilities for major actively supported releases:

| Version | Supported |
| :--- | :---: |
| v1.x.x | **Yes** |
| v0.x.x | No |

---

## Reporting a Vulnerability
We take the security of DairySphere very seriously. If you discover a potential vulnerability, please do NOT create a public GitHub Issue. Instead, report it privately:

1.  **Email:** Send an encrypted or plain-text email containing structural repro steps to `security@dairysphere.com`.
2.  **Acknowledge:** A security coordinator will acknowledge receipt of your vulnerability report within **24 hours**.
3.  **Resolution:** We aim to provide a complete software patch for all validated critical exploits within **48 hours** of successful reproduction.
