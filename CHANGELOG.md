# Changelog

All notable changes to the DairySphere project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-07-11

### Added
- Initialized core repository structure with enterprise standards.
- Added strict Prettier configuration (`.prettierrc` & `.prettierignore`).
- Added robust ESLint rules for TypeScript & React (`.eslintrc.json`).
- Configured Git properties (`.gitattributes`, `.editorconfig`).
- Integrated conventional commit structures via `commitlint.config.js`.
- Configured pre-commit validation and commit message validation via **Husky** hooks and `lint-staged`.
- Created unified project documentation (`README.md`, `LICENSE`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`).
